# 目录说明
- `tutorials`为课程文件夹，里面的格式说 `日期_更新序号.md`，如`20171117_1st.md`。
- `homeworks`为作业文件夹，使用说明见 `homework/README.md`。
- `src`为图片或相关引用文档目录

# 其他相关
* 小组问题可以新建`issue`讨论
* 请关注每次作业的`deadline`，毕竟老师们不能随时有提交，随时有时间检查。
